# Course Notebooks

This folder contains Jupyter notebooks covering theory and practical signal-processing exercises.

## Topics
- sensor fundamentals and signal chains
- measurements and basic metrology
- resistive sensors
- sampling and ADCs
- digital filters
- Kalman filters

## Running the notebooks

The notebooks are intended to be run using the conda environment provided in the repository root.

See the main setup instructions in the [repository README](../README.md#setup)
